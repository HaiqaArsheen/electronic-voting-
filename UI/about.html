CONSENSUS MECHANISM

In the POW consensus mechanism nodes compete to solve a complex mathematical puzzle to validate and add a new block.

CONSENSUS MECHANISM

Solving the puzzle requires significant computational power, ensuring that miners put in effort before adding block. POW is a decentralized consensus mechanism that requires network members to expand effort in solving an encrypted hexadecimal number. POW also called MINING, in reference to receiving a reward for work done. POW allow for secure peer-to-peer transaction processing without needing a trusted third party.
